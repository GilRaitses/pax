# Multi-Agent Work Trees

This directory contains work trees for organizing parallel work across multiple agents or work sessions.

## Files

- **MULTI_AGENT_WORK_TREE.md** - Main work tree with 5 branches for image processing tasks
- **AGENT_ASSIGNMENT_TEMPLATE.md** - Template for assigning work to agents

## How to Use

1. **Review the work tree:** Read `MULTI_AGENT_WORK_TREE.md` to understand available tasks
2. **Assign agents:** Use `AGENT_ASSIGNMENT_TEMPLATE.md` to assign branches to agents
3. **Track progress:** Update task checkboxes as work progresses
4. **Coordinate:** Use dependencies to coordinate between agents
